EzI2C Waveform Viewer
--------------

This project demonstrates the capabilites of the unique 'EzI2C' component in PSoC3/5.

The code pulls analog data from a pin,converts it to digital and updates the EzI2C buffer with it.This data can be viewed in the Bridge Control Panel,in streaming mode as a 'Waveform'.

Tested with PSoC Creator 2.0 and Bridge Control Panel 1.4 on the CY8CKIT-030.
